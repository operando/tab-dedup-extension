# Tab Dedup

開いているタブの中から URL が重複しているものを検出し、1つだけ残して残りを自動で閉じる Chrome 拡張です(Manifest V3)。

## インストール方法

1. Chrome で `chrome://extensions` を開く
2. 右上の「デベロッパーモード」を ON
3. 「パッケージ化されていない拡張機能を読み込む」を押す
4. このディレクトリ (`tab-dedup-extension`) を選択

## 使い方

ツールバーの拡張アイコンをクリックするとポップアップが開きます。

- **スキャン**: 現在のタブから重複候補をリスト表示します(閉じません)
- **重複を閉じる**: 重複を検出し、各 URL ごとに1タブだけ残して残りを閉じます

オプション:

- **URL のハッシュ (#...) を無視する**: 既定 OFF。同じページ内のアンカー違いを同一視
- **URL のクエリ (?...) も無視する**: 既定 OFF。トラッキングパラメータ等を含む URL を同一視
- **現在のウィンドウのみ対象**: 既定 OFF。OFF だと全ウィンドウが対象

## 残すタブの選び方

各重複グループから次の優先順で1タブを残します。

1. アクティブなタブ
2. ピン留めされたタブ
3. それ以外は最も古いタブ(タブIDが小さいもの)

## 対象 URL

`http(s)://`, `file://`, `ftp://` のみを対象にしています(`chrome://` などの内部ページは無視)。

## 履歴

「重複を閉じる」で閉じた URL は `chrome.storage.local` に履歴として保存されます。`unlimitedStorage` 権限を付与しているため件数の上限はありません(ディスク残量の許す限り)。

ポップアップの「閉じたURLの履歴を見る」ボタンから履歴ページを開けます。履歴ページでは次の操作が可能です。

- 検索ボックスで URL / タイトル絞り込み
- 各エントリを新しいタブで開く / URL をコピー
- JSON でエクスポート
- 履歴を全削除

履歴はローカルにのみ保存され、外部には送信されません。
